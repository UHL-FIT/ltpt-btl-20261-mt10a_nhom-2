BÁO CÁO TIẾN ĐỘ DỰ ÁN

  • Đề tài: Ứng dụng Real-time Currency Tracker Pro
  • Môn học: Lập trình Python
  • Nhóm thực hiện: Nhóm 2 (Trần Trung Hiếu,...)
  • Giảng viên hướng dẫn: Vũ Duy Sơn

1. Mục tiêu dự án

  • Xây dựng một ứng dụng Desktop bằng Python giúp theo dõi tỷ giá ngoại tệ trực tuyến.
  • Áp dụng các kiến thức cốt lõi: Lập trình hướng đối tượng, xử lý đa luồng (Multi-threading), phân tích dữ liệu (Pandas) và giao diện đồ họa (CustomTkinter).
  • Tuân thủ chặt chẽ mô hình thiết kế phần mềm MVC (Model - View - Controller).

2. Đã làm được gì? (Tổng quan)

  • Hoàn thành 100% Core System: Xây dựng xong bộ khung ứng dụng.
  • Cấu trúc thư mục chuẩn MVC: Phân tách rõ ràng Model (Dữ liệu), View (Giao diện), Controller (Điều khiển).
  • Tích hợp cơ sở dữ liệu nội bộ SQLite để lưu trữ lịch sử tỷ giá vĩnh viễn.
  • Hoàn thiện tài liệu kiến trúc (SAD.md), tài liệu yêu cầu (SRS.md) và đẩy code lên Github Classroom thành công.

3. Đã làm được gì? (Tính năng)

  • Thiết kế giao diện Dark Mode chuyên nghiệp, thân thiện.
  • Quản lý danh sách các mã ngoại tệ chuẩn ISO (Thêm/Sửa/Xóa).
  • Tích hợp thư viện Pandas tính toán: Giá trung bình, cao nhất, thấp nhất và độ biến động tiêu chuẩn.
  • Hỗ trợ xuất dữ liệu ra định dạng file .csv.
  • Đóng gói thành công phần mềm ra file .exe chạy độc lập.

4. Khó khăn gặp phải (Kỹ thuật)

  • Khó khăn: Giao diện ứng dụng bị đơ (not responding) khi gọi API lấy dữ liệu trên luồng chính do mạng chậm.
  • Cách khắc phục: Nhóm đã nghiên cứu sử dụng thư viện threading, đưa tác vụ gọi API xuống chạy ngầm (Background Worker) và dùng cơ chế đồng bộ hóa để cập nhật lên màn hình an toàn.

5. Khó khăn gặp phải (Quy trình)

  • Khó khăn: Gặp hiện tượng xung đột code (Merge Conflicts) khi tải file bài mẫu của thầy từ Github về máy. Lỗi PyInstaller không đóng gói được do vướng conflict markers.
  • Cách khắc phục: Chạy lệnh checkout xử lý xung đột của Git, sửa tay file main.py và cấu hình lại môi trường để PyInstaller build thành công.

6. Kế hoạch tiếp theo (Tối ưu hóa)

  • Tinh chỉnh lại các khoảng cách (padding/margin) của giao diện để phù hợp trên nhiều kích thước màn hình máy tính khác nhau.
  • Tối ưu hoá bộ nhớ (RAM) khi tải hàng nghìn bản ghi lịch sử tỷ giá vào biểu đồ Matplotlib.
  • Bổ sung hiệu ứng chuyển động mượt mà hơn cho Splash Screen (Màn hình chờ).

7. Kế hoạch tiếp theo (Tính năng mở rộng)

  • Tính năng Cảnh báo (Alert): Tự động đổi màu văn bản thành màu đỏ/cam khi tỷ giá của một cặp tiền vượt quá một ngưỡng nhất định.
  • Mini-Calculator: Thêm một công cụ máy tính nhỏ ở thanh Sidebar để người dùng nhập số tiền và đổi trực tiếp theo tỷ giá thực mà không cần mở máy tính ngoài.
